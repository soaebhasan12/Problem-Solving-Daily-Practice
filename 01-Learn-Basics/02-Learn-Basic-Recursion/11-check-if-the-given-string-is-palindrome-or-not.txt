# 🧠 1️⃣ Core Concept Insight

Palindrome check = **Symmetric comparison problem**

Signal:

* Compare first ↔ last
* Move inward
* Stop at middle

Whenever problem says:

* Same forward and backward
* Symmetric
* Mirror property

→ Think **Two Pointer Inward Pattern**

# 🧠 3️⃣ Most Important Pattern Learned

If problem says:

“Ignore some characters”

Flow must be:

Skip invalid → Compare valid → Move inward

Order matter karta hai.

Wrong order → logic break.

---

# 🧠 4️⃣ Pointer Movement Insight

Loop condition:

left < right

Even length:
Pointers cross.

Odd length:
Pointers meet at middle.

Middle character automatically handled.

---

# 🧠 5️⃣ Common Bugs You Faced

❌ Return True inside loop
❌ Not moving pointers after valid match
❌ Comparing before skipping invalid
❌ Wrong logical block order

These are classic two-pointer mistakes.

---

# 🧠 6️⃣ Time & Space Insight

Time Complexity → O(n)
Each character visited at most once.

Space Complexity → O(1)
Because no extra string required.

Signal:

If you can solve without building new structure → optimize space.

# 🧠 7️⃣ Pattern Recognition Signals

Use Two Pointers when:

* Check symmetry
* Reverse something
* Compare from both ends
* Trim / skip while scanning inward
